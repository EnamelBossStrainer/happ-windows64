**Happ** — приложение для работы с прокси-подключениями, основанное на ядре **Xray**. Оно предоставляет удобные инструменты для создания, импорта и управления конфигурациями, поддерживает современные сетевые протоколы и гибкие правила маршрутизации. Лаконичный интерфейс делает работу с приложением простой как для начинающих пользователей, так и для тех, кто регулярно использует прокси в повседневных задачах.

## Что умеет Happ

* Создание, импорт и изменение прокси-конфигураций.
* Настройка маршрутизации трафика по собственным правилам.
* Поддержка популярных протоколов:

  * **VLESS (Reality)**
  * **VMess**
  * **Trojan**
  * **Shadowsocks**
  * **SOCKS**
* Быстрое переключение между сохраненными конфигурациями.
* Централизованное управление всеми подключениями.

## Конфиденциальность

Happ не собирает пользовательскую статистику, не отправляет конфигурации на внешние серверы и не передает личные данные третьим лицам. Все настройки, профили и служебная информация хранятся исключительно на устройстве пользователя.

## Важно

Happ является инструментом для управления прокси-конфигурациями и **не предоставляет VPN-сервис, прокси-серверы или подписку на них**. Для работы необходимо использовать собственный сервер либо инфраструктуру стороннего поставщика. Использование приложения должно соответствовать законодательству страны, в которой находится пользователь.
